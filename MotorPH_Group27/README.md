# MotorPH Employee Salary Computation System

## Overview

The MotorPH Employee Salary Computation System is a Java Swing-based desktop application designed to manage employee records and perform salary computations. The system uses CSV files for data storage and provides a user-friendly interface for managing employee information.

## Features

### Feature 1 – Graphical User Interface (GUI)

* Java Swing user interface
* Employee records displayed using JTable
* Employee summary panel
* User-friendly navigation and controls

### Feature 2 – Employee Management

* View employee records
* Add new employees
* Display employee details
* CSV-based data storage

### Feature 3 – Salary Computation

* Gross pay calculation
* SSS contribution computation
* PhilHealth contribution computation
* Pag-IBIG contribution computation
* Withholding tax computation
* Net pay computation
* SalaryResults.csv generation

### Feature 4 – Employee Record Maintenance

* Update employee information
* Delete employee records
* Edit personal information
* Edit government identification numbers
* Edit employee status and position
* Persistent CSV file updates

## Technologies Used

* Java
* Java Swing
* ArrayList
* CSV File Handling
* NetBeans IDE

## Files

* Employee.java
* EmployeeManager.java
* EmployeeDataHandler.java
* SalaryComputationModule.java
* GUI.java
* Main.java
* EmployeeDetails.csv

## How to Run

1. Open the project in NetBeans IDE.
2. Ensure EmployeeDetails.csv is located in the project directory.
3. Run Main.java.
4. Use the GUI to manage employee records and compute salaries.
